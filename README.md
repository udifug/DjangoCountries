Как запустить проект после клонирования
1. source countries_venv/Scripts/activate  
2. pip install -r requirements.txt
3. python manage.py migrate
Загрузка фикстур в бд:
4. python manage.py loaddata MainApp/fixtures/countries.json
   python manage.py loaddata MainApp/fixtures/languages.json
5. python manage.py runserver
